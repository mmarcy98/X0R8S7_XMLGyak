Adatkezelés XML-ben

Az XML alapú adatmodellezés elveinek elsajátítása, a XML adatok kezelési szabványainak megismerése. Az XMLSchema, a DOM és az XSLT és XQuery felületek biztos programozási készségének elsajátítása.
A XML feldolgozó API-k (SAX, DOM, JAXB) megismerése és gyakorlása. A kapcsolódó szabványok áttekintése (JSON, SVG,..).


Tartalom:
  • XML alapjai
  • XMLSchema
  • DOM
  • XSLT
  • XQuery
  • XML API-k
  • Kapcsolódó szabványok
